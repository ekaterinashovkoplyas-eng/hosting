Interior Design Icons

Иконки для эстетики интерьера

Чистый HTML-шаблон с набором интерьерных пиктограмм: мебель, свет, декор и планировка. Подходит для портфолио студии, каталога услуг или презентации дизайн-системы.

Коллекция

6

Минималистичные SVG-иконки встроены прямо в страницу и масштабируются без потери качества.

01 / Furniture

Диван

Иконка мягкой мебели для разделов о гостиных, подборе гарнитуров и moodboard-панелей.

02 / Lighting

Лампа

Легкая графика для блоков об освещении, сценариях света и декоративных подвесах.

03 / Decor

Растение

Подходит для категорий биофильного дизайна, зелёного декора и акцентных интерьерных деталей.

04 / Dining

Стол

Символ обеденной зоны или рабочего пространства для планировок, лендингов и карточек проектов.

05 / Art

Постер

Иконка настенного декора для галерей, каталогов искусства и презентации стилистических решений.

06 / Layout

Планировка

Универсальная пиктограмма для чертежей, схем помещений и концептов по зонированию пространства.

Страница не использует внешние библиотеки: откройте файл в браузере и при необходимости замените тексты, палитру или SVG на собственный набор иконок.
